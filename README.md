# 《活法》要点导读 · 稻盛和夫

稻盛和夫《活法》核心要点整理网页：人生方程式、六项精进、工作即修行、利他与判断基准。

## 本地预览

用浏览器直接打开 `index.html`，或在本目录启动静态服务：

```bash
# Python
python -m http.server 8080

# Node
npx --yes serve .
```

然后访问 `http://localhost:8080`。

## GitHub Pages

仓库启用 Pages 后（Settings → Pages → Deploy from branch `main` / `/root`），访问：

`https://<你的用户名>.github.io/huofa-summary/`
